Команда cat

просмотр содержимого файла
$ cat file
Команда просто выведет все, что есть в файле. 

Чтобы вывести несколько файлов достаточно просто передать их в параметрах:
$ cat file file1

стандартный поток ввода можно обозначить с помощью символа "-". Поэтому мы можем комбинировать вывод текста из файла, а также стандартного ввода:
$ cat file - file1

чтобы нумеровать только непустые строки используйте:
$  cat -b file

нумеровать все строки в файле:
$  cat -n file

позволяет удалить повторяющиеся пустые строки:
$ cat -s file

сообщить утилите, что нужно отображать символ $ в конце каждой строки:
$ cat -E file 
(у меня почему-то так:
$  cat -E file1.txt
$ривет
$
$ак дела
$
Понедельник)

Таким образом можно получить очень примитивный текстовый редактор - прочитаем ввод и перенаправим его вместо вывода на экран в файл:
$ cat file1 > file2
$ cat file2
(в file2 оказался file1)

$ cat > file2
$ cat file2
(в файле file2 пусто)

Возможность объединения нескольких файлов не была бы настолько полезна, если бы нельзя было записать все в один:
$ cat file1 file2 > file3
$ cat file3
(в file3 объединены два файла 1 и 2)


Если вы не передадите никакого файла в параметрах утилите, то она будет пытаться читать данные из стандартного ввода:
$ cat
cat
(пропал $ - ничего нельзя запрашивать)
Для завершения записи нажмите Ctrl+D.
