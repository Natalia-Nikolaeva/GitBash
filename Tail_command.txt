Команда Tail
Отображает последнюю часть (по умолчанию 10 строк) одного или нескольких файлов или передаваемых дан-ных. Можно использовать для отслеживания изменений файлов в реальном времени.
Основной синтаксис:
tail [OPTION]... [FILE]...
OPTION — варианты хвоста . 
FILE — Ноль или более имен входных файлов. Если ФАЙЛ не указан или если ФАЙЛ равен - , tail будет читать стандартный ввод.
tail file.txt отобразить последние 10 строк.
tail -n <NUMBER> file.txt указать количество отобража-емых строк. 
tail -n 50 file.txt отобразить последние 50 строк файла
>>Можно опустить букву n и использовать только дефис ( - ) и цифру (без пробелов между ними)<<
tail -c <NUMBER> file.txt показать определенное количе-ство байтов
tail -f file.txt отслеживать изменения в файле
tail -f /var/log/nginx/error.log опция особенно полезна для мониторинга файлов журналов. Здесь чтобы отоб-разить последние 10 строк фай-ла /var/log/nginx/error.log и отслеживать обновления файла.
Ctrl+C  прервать команду tail во время просмотра файла
tail -F file.txt продолжить наблюдение за файлом при его воссоздании
tail file1.txt file2.txt отобразить последние десять строк из каждого файла
tail -n 20 file1.txt file2.txt показать последние 20 строк файлов
tail может использоваться в сочетании с другими коман-дами путем перенаправления стандартного вывода из / в другие утилиты с помощью каналов.
